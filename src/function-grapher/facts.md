# function-grapher 知识断言

## 适用范围

- 课件研究实变量函数 y=ax+b、y=ax²+bx+c、y=k/x 和 y=a·sin(bx+c)；正弦函数的自变量与相位采用弧度。
- 画布只是约 [-10.5,10.5]² 的有限视窗与有限采样近似，不代表完整定义域、值域，也不能代替数学证明。
- “一次函数”“二次函数”“反比例函数”分别要求 a≠0、a≠0、k≠0；参数为 0 时课件按退化函数另行说明。

## 知识断言

- [F1] 函数给定义域中的每个输入唯一确定一个输出；函数图像由所有满足 y=f(x) 的有序对组成。[S1] [S5]
- [F2] 对 y=ax+b，a 是斜率，图像经过 (0,b)；a≠0 时还经过 (−b/a,0)。按中国教材术语，a≠0 时才是一次函数；a=0 时是常函数 y=b。[S2] [S5]
- [F3] 二次函数 y=ax²+bx+c 要求 a≠0；其对称轴为 x=−b/(2a)，顶点为 (−b/(2a),f(−b/(2a)))；a>0 时开口向上，a<0 时开口向下，其余参数固定时 |a| 增大，图像变窄。[S3] [S6]
- [F4] 反比例函数 y=k/x 要求 k≠0，定义域和值域均为 ℝ∖{0}；k>0 时图像位于第一、三象限，k<0 时位于第二、四象限，x=0 与 y=0 是渐近线。[S1] [S6]
- [F5] 对课件范围 b>0 的 y=a·sin(bx+c)，定义域为 R，值域为 [−|a|,|a|]；a≠0 时振幅为 |a|、最小正周期为 2π/b，相对 y=a·sin(bx) 的水平平移量为 −c/b；a=0 时为零函数，没有最小正周期。[S4] [S6]
- [F6] 图像变换规律必须附带“其余参数固定”的条件；有限滑块示例只用于观察特定参数值，不等同于证明规律对所有参数成立。[S4] [S5]
