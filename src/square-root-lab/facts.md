# square-root-lab 知识断言

## 适用范围

- 课件只在实数范围内计算非负数的算术平方根；负数的平方根不在本课件讨论范围内。
- 根式化简仅用于非负整数，并通过提出完全平方因子完成；小数结果是有限位数近似，不冒充精确值。
- 面积图用于建立“正方形面积 ↔ 边长”的直观联系，不代替平方根存在性或无理性的数学证明。

## 知识断言

- [F1] 对任意实数 a≥0，存在唯一的非负实数 x 使 x²=a；这个 x 记作 √a，称为 a 的算术平方根（主平方根）。[S1] [S2]
- [F2] 根号 √a 表示非负的主平方根。例如 √9=3；而方程 x²=9 有两个实数解 x=3 与 x=−3，二者不可混淆。[S1] [S3]
- [F3] 若 n=k²m，且 k、m 为非负整数，则 √n=k√m；因此 √72=√(36×2)=6√2。[S2] [S3]
- [F4] 若非负整数 n 不是完全平方数，且 k²<n<(k+1)²，则 k<√n<k+1；例如 8²<72<9²，所以 8<√72<9。[S1] [S2]
- [F5] 小数显示值由算术平方根四舍五入得到。例如 √72≈8.4853 是保留四位小数的近似值，不是精确等式。[S2]
