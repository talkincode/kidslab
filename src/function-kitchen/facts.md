# function-kitchen 知识断言

## 适用范围

- 面向小学五至六年级的函数概念入门，使用无返回值的命名配方模拟 JavaScript 函数。
- 覆盖函数封装、重复调用和形参/实参的直观对应；不覆盖作用域、闭包、递归或 JavaScript 语法细节。

## 知识断言

- [F1] 函数可以把一组执行任务的语句封装起来，并通过调用重复执行这组语句。[S1]
- [F2] 函数定义中的参数是接收输入的名称；调用函数时传入的实参会按位置对应到这些参数。[S1]
- [F3] 修改函数定义中的共享步骤，会影响之后使用该定义进行的调用，而不必逐份修改重复代码。[S1]

## 来源

- [S1] MDN Web Docs, “Functions — JavaScript Guide”, Function declarations and Function parameters: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Functions
