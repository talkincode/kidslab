# fraction-lab 知识断言

## 适用范围

- 课件只覆盖整数分子、分母，且 1 ≤ 分母 ≤ 12、0 ≤ 分子 ≤ 分母。
- 课件使用一个整体的部分—整体模型，不覆盖假分数、带分数、负分数或分数运算。

## 知识断言

- [F1] 把一个整体平均分成 d 份，取其中 n 份，可以表示为 n/d；分母 d 不能为 0。[S1] [S2]
- [F2] 在课件范围内，未取部分有 d−n 份；0/d=0，d/d=1。[S1]
- [F3] 等面积披萨扇形、等大巧克力格和数轴位置可以表示同一个数 n/d；因为 0≤n≤d，该数位于闭区间 [0,1]。[S1]
- [F4] n/d=n÷d，百分数为 (n÷d)×100%；结果不能有限表示时，课件必须明确标为近似值。[S1] [S2]
- [F5] 对非零整数 k，n/d=(n×k)/(d×k)；课件只展示 k=1、2、3。[S2]
- [F6] 若 n、d 的最大公因数为 g>1，则 n/d=(n÷g)/(d÷g)，约分后分子与分母互质，得到最简分数。[S2]
